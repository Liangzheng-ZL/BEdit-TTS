# Samples

## Model

**Baseline 1:** This system is based on  FastSpeech2 model, which generates a complete speech audio from the whole the edited text.

**Baseline 2:** This system is also based on FastSpeech2 model, which generates the audio segment of the modified region and inserts the generation back to the original speech.

**Baseline 3:** This system is based on FastSpeech 2 model, but generates a complete speech audio from the whole edited text, and the audio segment of the modified region is cut from the whole generation, then the audio segment is inserted back to the original speech with alignment information.
